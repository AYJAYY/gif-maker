# Framezy

Turn any video playing on your screen into a GIF — without a separate
screen-recorder-then-crop workflow. Drag a transparent frame directly over
the video, record, trim, and export.

## How it works

1. Open Framezy — a small control panel appears.
2. Click **Select Region** — a frame appears with a transparent, click-through
   center. Drag it over the video and resize using the corner/edge dots. You
   can still click play/pause on the video underneath, right through the hole.
3. Click **Record**.
4. Click **Stop** — a trim/preview editor opens. Cut it down, set fps, size,
   speed, loop count, or add a reverse/boomerang effect.
5. Click **Export GIF** and save.

## Download

Grab the latest installer or portable `.exe` from the
[Releases page](https://github.com/AYJAYY/gif-maker/releases/latest):

- **FramezySetup.exe** — installer, adds a Start Menu shortcut and uninstaller.
- **Framezy.exe** — portable, just run it, no install step.

Requires Windows 10 or 11.
